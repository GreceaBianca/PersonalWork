A short summary of the apps:
 - coffeeapp: application for ordering coffee with Vue
 - heroapp: application to manage heroes, sidekicks and planets to save the world with Angular and C#. Data is stored in a database.
 - rentPrediction: the application has a scraper that gathers properties from the web and uses it to train the ML engine which is later used in the interface to help users see if the price for the property is fair or not. The prediction is made using essentials features like location, surface, number of rooms, parking and many other.
